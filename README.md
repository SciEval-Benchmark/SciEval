# SciEval

A benchmark dataset for Automatic Evaluation of K-12 Science Instructional Materials.

**Website:** (https://freeman-gif.github.io/SciEval/)

## Dataset

SciEval contains 273 lesson-level instructional materials annotated with pedagogy-aligned evaluation scores and evidence-based rationales across 13 EQuIP rubric criteria (N=3,549).

Download the dataset from the [Releases](https://github.com/Freeman-gif/SciEval/releases) page.
